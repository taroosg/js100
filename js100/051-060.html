プロフィールの表示

プロフィールの形式は「彼/彼女の名前はnameでage歳です」とします．彼/彼女はgenderによって区別されるものとします．

3項目揃っていない場合は「情報がありません」と出力してください．

}

  name : 'Taro',
  age : 25,

};

  name : 'Hanako',
