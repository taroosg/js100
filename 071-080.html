フィボナッチ数列

自然界に多くみられる数列として，フィボナッチ数列という数列があります．フィボナッチ数列は以下の公式で表されます．

【フィボナッチ数列とは】

  • f(0) = 0
  • f(1) = 1
  • f(n) = f(n-2) + f(n-1) (ただし，n >= 2であることが条件)

【具体例】

  • f(2) = 0 + 1 = 1
  • f(3) = 1 + 1 = 2
  • f(4) = 1 + 2 = 3
  • f(5) = 2 + 3 = 5
